# RxJS Pong

[Pong](https://de.wikipedia.org/wiki/Pong) using RxJS

```
npm install
npm start
```

## Steps
1. game board with nothing moving (boarder, paddles, middle line)
2. move the ball
3. bounce on collision
4. move the paddles
5. bounce only on wall and paddle, otherwise point for one player
6. sound
